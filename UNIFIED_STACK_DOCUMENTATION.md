## Installation & Setup

### Prerequisites
- Windows 10/11
- Python 3.8+
- Administrator privileges (for system operations)

### Required Python Packages
```
psutil>=5.8.0
tkinter (included with Python)
```

### Installation Steps
1. Ensure all core files are present
2. Run verification: `python VERIFY_SIMPLE.py`
3. Launch system: `LAUNCH_UNIFIED_STACK.bat`

- Run as administrator

#### Performance Issues

- Run system optimization
