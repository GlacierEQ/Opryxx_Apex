# OPRYXX User Guide

## Quick Start

### 1. Installation
```cmd
# Install dependencies
INSTALL_DEPENDENCIES.bat

# Build EXE files (optional)
BUILD_EVERYTHING.bat
```

### 2. Launch Systems
```cmd
# Master control panel
MASTER_LAUNCHER.bat

# Individual systems
LAUNCH_ULTIMATE_AI.bat      # 24/7 AI optimizer
LAUNCH_AI_WORKBENCH.bat     # AI health manager
LAUNCH_ENHANCED_AI.bat      # Enhanced AI with notifications
```

## Troubleshooting

### Common Issues

**Python Not Found**
- Install Python 3.8+ from python.org
- Add Python to system PATH

**Permission Errors**
- Run as Administrator
- Check Windows UAC settings

**Missing Dependencies**
- Run `INSTALL_DEPENDENCIES.bat`
- Manually install: `pip install psutil`

**Safe Mode Issues**
- Use `EMERGENCY_RECOVERY.bat`
- Select "Immediate Safe Mode Exit"
- Reboot system

### Performance Issues

**Low Performance Score**
- Run memory optimization
- Check for background processes
- Update GPU drivers

**Memory Leaks Detected**
- NEXUS AI auto-fixes leaks
- Manual fix: restart problematic applications
- Check system RAM usage
